## Setting up neovim ##

1. ``` sudo apt-get install neovim ```

2. ``` sudo apt-get install curl ```

3. ``` sudo apt-get install git ````

4. ``` mkdir ~/.config/nvim   ```

5. ``` touch ~/.config/nvim/init.vim   ```

6. ``` nvim ~/.config/init.vim  ```
